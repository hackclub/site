# Hack Club для IT-администраторов
Если вы читаете это, вы, вероятно, IT-администратор в своей организации. **Вот краткий обзор того, что нам от вас нужно.**

Пожалуйста, внесите в белый список (whitelist) или разблокируйте следующие домены:

- `*.hackclub.com` — основной домен Hack Club; на нём размещены воркшопы, инструменты и другой контент
- `hack.af` — сокращатель ссылок для различных ссылок
- `*.github.com`, `github.dev` — GitHub, отраслевой стандарт хостинга кода и совместной работы, принадлежит Microsoft
- `hackclub.slack.com`, `hackclub.enterprise.slack.com` — основная платформа общения Hack Club

Ниже — дополнительные детали и возможные альтернативы.

## Сайты Hack Club

Если вы не можете разрешить все поддомены, разрешите следующие:

- `hackclub.com`
- `workshops.hackclub.com`
- `jams.hackclub.com`
- `sprig.hackclub.com`

## GitHub

Помимо обычных портов HTTP(S), для SSH и Git должны быть открыты порты 22 и 9418 соответственно.

## Inspect Element / Инструменты разработчика

Инструменты разработчика в браузере (например, Inspect Element) необходимы для веб-проектов — чтобы отлаживать и стилизовать сайты.
